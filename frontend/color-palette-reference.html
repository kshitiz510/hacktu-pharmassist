Best Practices

  • Consistency: Use PRIMARY_PALETTE for all standard charts (bar, pie, line)
  • Distinction: Use EXTENDED_PALETTE when you have 8+ data categories
  • Gradients: Always apply lightenColor() for gradient fills to add depth
  • Accessibility: Ensure color contrasts meet WCAG AA standards
  • Monochromatic: Use scales for heatmaps and value-based coloring
  • Alternation: Use getAlternatingColors() to maximize visual distinction
